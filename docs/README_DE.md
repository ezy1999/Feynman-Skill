# Feynman-Skill: Feynmans Forschungsgeschmack-Modellierungssystem

> **Evidenzbasierte Modellierung von Richard Feynmans wissenschaftlichem Forschungsgeschmack.**

## Was ist das?

Dieses System modelliert Feynmans **Forschungsgeschmack** - seine einzigartigen Praferenzen fur bestimmte wissenschaftliche Ansatze.

**Dies ist KEIN Rollenspiel.** Jede Bewertung basiert auf historischen Belegen.

## Die 10 Geschmachsachsen

| # | Achse | Gewicht | Bedeutung |
|---|-------|---------|-----------|
| 1 | **Physische Intuition** | 0.95 | Bilder und Visualisierung vor Formalismus |
| 2 | **Rechnerischer Pragmatismus** | 0.90 | Kann man eine Zahl berechnen? |
| 3 | **Empirische Strenge** | 0.90 | „Wenn es nicht mit dem Experiment ubereinstimmt, ist es falsch" |
| 4 | **Spielerische Erforschung** | 0.85 | Neugiergetrieben, ohne Druck |
| 5 | **Unabhangiges Denken** | 0.85 | Autoritat hinterfragen, erste Prinzipien |
| 6 | **Anti-Formalismus** | 0.80 | Misstrauen gegenuber reiner Abstraktion |
| 7 | **Bottom-Up-Denken** | 0.80 | Von Beispielen ausgehen, dann verallgemeinern |
| 8 | **Mehrfachdarstellung** | 0.75 | Mehrere Sichtweisen = tieferes Verstandnis |
| 9 | **Einfachheit der Erklarung** | 0.75 | „Wenn Sie es einem Erstsemester nicht erklaren konnen..." |
| 10 | **Interdisziplinare Vielseitigkeit** | 0.70 | Ideen zwischen Fachgebieten ubertragen |
